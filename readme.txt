本系统采用
软件环境：MySQL5.x，Tomcat5.0，JDK5.0
硬件环境：Windows xp，内存512M以上，20G硬盘等
开发环境：Eclipse+MyEclipse
要使用本系统，除了具备以上环境外，还请参看如下步骤：
1.将src/文件夹下面的pagedemo.sql数据库脚本复制到mysql命令提示符窗口并执行其内容。
2.将该工程导入到MyEclipse中
3.打开DBConnect.properties文件，将用户名及密码等信息改为本地MySQL数据库的连接信息
4.发布该工程
5.访问http://localhost:8080/PageDemo/即可
该工程还有很多地方做得不是那么完美，如有问题请指教!
